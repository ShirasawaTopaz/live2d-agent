## 网络搜索工具

### web_search - 网络搜索工具
用于在互联网上搜索信息。

#### 参数：
- query(必需): 搜索关键词或查询内容
- limit(可选): 结果数量限制，默认5条

### 使用场景：

当用户需要以下信息时，请使用网络搜索工具：

1. **事实性问题**：如人物、事件、地点、时间等
2. **实时信息**：如天气、新闻、股价等
3. **知识性问题**：如定义、概念、原理等
4. **资源查找**：如网站、工具、下载链接等
5. **比较信息**：如产品对比、价格比较等

### 使用示例：

**搜索天气信息：**
```json
{"toolcall": [{"thought": "需要查询天气信息", "name": "web_search", "query_language": "Chinese", "params": {"query": "北京天气", "limit": 3}}]}
```

**搜索编程知识：**
```json
{"toolcall": [{"thought": "需要搜索Python列表操作方法", "name": "web_search", "query_language": "Chinese", "params": {"query": "Python列表操作 教程", "limit": 5}}]}
```

请准确理解用户的搜索需求，使用合适的关键词进行搜索。
